# STL Importer
STL import plugin for Godot.

## Downloading and activating the Plugin
1. Download the Plugin using AssetLib.
1. Click on Project / Project Settings to open the Project Settings Menu.
1. Click on Plugins and make sure "STL Importer" is activated.

That's it, your existing STL-files will then be imported, you can also simply drag-and-drop the files to your Godot window and then into your scene

Note that most STLs are Z-up by and in Millimetre by default for mechanics, robotics and 3D-printing. This means you probably want to rotate the STL by **-90°** degree and scale it down to **0.001**.
